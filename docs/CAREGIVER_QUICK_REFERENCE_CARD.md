# PILDHORA Caregiver Quick Reference Card

## 🚀 Quick Start

### First Time Setup (5 Minutes)
1. Download app → Sign up → Select "Caregiver"
2. Get device ID from patient
3. Tap "Device" → "Link New Device" → Enter ID
4. Start managing medications!

## 💊 Medication Management

### Add Medication
1. Tap "Medications" → "+"
2. Choose icon & name
3. Set dosage & unit
4. Add schedule times
5. Enable inventory (optional)
6. Save

### Edit Medication
Tap medication → "Edit" → Make changes → "Save"

### Delete Medication
Tap medication → "Delete" → Confirm

---

## 🔔 Events Registry

### View Events
Tap "Events Registry" → See all medication activities

### Filter Events
Tap "Filters" → Select:
- Date range
- Event type
- Medication name
- Patient

### Generate Report
Menu (⋮) → "Generate Report" → Choose PDF/CSV → Share

---

## 📱 Device Management

### Check Status
Dashboard shows:
- 🟢 Online / ⚫ Offline
- 🔋 Battery level
- ⏰ Last seen

### Configure Device
1. Tap "Device" → Find device → "Configure"
2. Set alarm mode, volume, LED
3. Choose LED color
4. Save

### Link New Device
"Device" → "Link New Device" → Enter ID → "Link"

---

## ✓ Tasks

### Add Task
"Tasks" → "+" → Type task → "Add"

### Complete Task
Tap checkbox → Task marked done

### Delete Task
Swipe left → "Delete"

## 🔍 Quick Troubleshooting

### Device Offline?
1. Check device power
2. Verify WiFi connection
3. Restart device
4. Wait 2 minutes

### Medication Not Syncing?
1. Ensure device is online
2. Wait a few minutes
3. Try editing medication again

### App Slow?
1. Close other apps
2. Restart app
3. Clear cache
4. Restart phone

### Can't Log In?
1. Check email/password
2. Try "Forgot Password"
3. Check internet connection
4. Contact support
