This guide will help you get started with PILDHORA in just a few minutes. Follow these simple steps to begin managing your patients' medications effectively.

## Quick Start (5 Minutes)

### Step 1: Download and Install (1 minute)

**iOS Users**:
1. Open the App Store
2. Search for "PILDHORA"
3. Tap "Get" to download
4. Wait for installation to complete

**Android Users**:
1. Open Google Play Store
2. Search for "PILDHORA"
3. Tap "Install"
4. Wait for installation to complete

---

### Step 2: Create Your Account (2 minutes)

1. **Open the App**
   - Tap the PILDHORA icon on your home screen

2. **Sign Up**
   - Tap "Sign Up" button
   - Enter your email address
   - Create a strong password (minimum 8 characters)
   - Tap "Continue"

3. **Select Your Role**
   - Choose "Caregiver" from the options
   - This gives you access to caregiver features

4. **Complete Your Profile**
   - Enter your full name
   - Add phone number (optional)
   - Upload profile photo (optional)
   - Tap "Complete Registration"

5. **Verify Your Email**
   - Check your email inbox
   - Open the verification email from PILDHORA
   - Click the verification link
   - Return to the app

**âœ… Account Created!**

---

### Step 3: Link Your First Patient (2 minutes)

1. **Get the Device ID**
   - Ask your patient for their device ID
   - Device ID is printed on the PILDHORA device
   - Example format: "DEVICE-001" or "PILDHORA-ABC123"

2. **Link the Device**
   - On the dashboard, tap "Device" in Quick Actions
   - Tap "Link New Device" button
   - Enter the device ID carefully
   - Tap "Link Device"

3. **Confirmation**
   - You'll see a success message
   - The patient appears in your dashboard
   - Device status shows online/offline

**âœ… First Patient Linked!**

## Your First Day

### Explore the Dashboard

**What You'll See**:
- **Header**: Your name and quick action buttons
- **Patient Selector**: Switch between patients (if you have multiple)
- **Device Status**: Real-time device connectivity and battery
- **Last Medication**: Most recent medication event
- **Quick Actions**: Four buttons for main features

**Try This**:
1. Tap each Quick Action button to explore
2. Check the device connectivity status
3. View the last medication event
4. Familiarize yourself with the layout

---

### Add Your First Medication

Let's add a medication for your patient:

1. **Start the Wizard**
   - Tap "Medications" in Quick Actions
   - Tap the "+" button at the bottom

2. **Choose Icon and Name**
   - Tap an emoji icon (e.g., ðŸ’Š for pills)
   - Enter medication name (e.g., "Aspirin")
   - Tap "Next"

3. **Set Dosage**
   - Enter dose amount (e.g., "100")
   - Select unit (e.g., "mg")
   - Choose type (e.g., "Pill")
   - Tap "Next"

4. **Create Schedule**
   - Select frequency (e.g., "Daily")
   - Tap "+" to add a time
   - Set time (e.g., "08:00 AM")
   - Add more times if needed
   - Tap "Next"

5. **Inventory (Optional)**
   - Toggle "Track Inventory" if desired
   - Enter current quantity
   - Set low quantity alert
   - Tap "Save"

**âœ… First Medication Added!**

The device will now alarm at the scheduled time.

---

### Check the Events Registry

See what's been recorded:

1. Tap "Events Registry" in Quick Actions
2. You'll see the "Medication Created" event
3. Tap the event to see details
4. Notice the timestamp and medication info

**What Events Show**:
- Medication created, updated, deleted
- Doses taken by patient
- Doses missed by patient

---

### Create Your First Task

Keep track of caregiver to-dos:

1. Tap "Tasks" in Quick Actions
2. Tap the "+" button
3. Enter task description (e.g., "Call pharmacy for refill")
4. Tap "Add"

**Task Management**:
- Check off tasks when complete
- Edit tasks by tapping them
- Delete tasks by swiping left

## Your First Week

### Day 1-2: Learn the Basics

**Goals**:
- [ ] Add 2-3 medications
- [ ] Check device status daily
- [ ] Review events registry
- [ ] Create 3-5 tasks

**Activities**:
1. Add all current medications for your patient
2. Verify medication schedules are correct
3. Check that device alarms are working
4. Review events to ensure everything is syncing

---

### Day 3-4: Explore Features

**Goals**:
- [ ] Configure device settings
- [ ] Set up notifications
- [ ] Try filtering events
- [ ] Generate a report

**Activities**:

**Configure Device**:
1. Tap "Device" in Quick Actions
2. Find your device and tap "Configure"
3. Adjust alarm settings:
   - Alarm Mode: Sound, Vibrate, Both, or Silent
   - Volume: Set comfortable level
   - LED Intensity: Adjust brightness
   - LED Color: Choose alarm color
4. Tap "Save Configuration"

**Set Up Notifications**:
1. Go to Settings (tap account icon)
2. Tap "Notifications"
3. Enable desired notifications:
   - Low battery alerts
   - Missed dose alerts
   - Low quantity alerts
   - Device offline alerts
4. Save settings

**Filter Events**:
1. Go to Events Registry
2. Tap "Filters"
3. Try different filters:
   - Date range
   - Event type
   - Medication name
4. Clear filters to see all events

---

### Day 5-7: Advanced Features

**Goals**:
- [ ] Link a second patient (if applicable)
- [ ] Edit a medication
- [ ] Track inventory
- [ ] Generate and share a report

**Activities**:

**Link Second Patient** (if applicable):
1. Get second device ID
2. Tap "Device" > "Link New Device"
3. Enter device ID and link
4. Use Patient Selector to switch between patients

**Edit Medication**:
1. Go to Medications
2. Tap a medication card
3. Tap "Edit"
4. Make changes (e.g., adjust dosage)
5. Tap "Save"
6. Check Events Registry for update event

**Track Inventory**:
1. Edit a medication
2. Enable "Track Inventory"
3. Enter current quantity
4. Set low quantity threshold
5. Save changes
6. You'll get alerts when quantity is low

**Generate Report**:
1. Go to Events Registry
2. Apply filters if desired
3. Tap menu icon (â‹®)
4. Select "Generate Report"
5. Choose format (PDF or CSV)
6. Share with doctor or save

## Best Practices

### Daily Routine

**Morning** (5 minutes):
1. Open PILDHORA app
2. Check device status for all patients
3. Review overnight events
4. Check for any missed doses
5. Review today's tasks

**Evening** (5 minutes):
1. Check device battery levels
2. Review day's medication events
3. Update tasks as needed
4. Prepare for next day

---

### Weekly Routine

**Once a Week** (15 minutes):
1. Review all medications for accuracy
2. Check inventory levels
3. Generate weekly report
4. Update medication schedules if needed
5. Clear completed tasks
6. Check for app updates

---

### Monthly Routine

**Once a Month** (30 minutes):
1. Generate monthly report for each patient
2. Review medication adherence
3. Update device configurations if needed
4. Review and optimize notification settings
5. Clear app cache
6. Back up important data

### Troubleshooting

**Common Issues**:

**Device Offline**:
- Check device power
- Verify WiFi connection
- Restart device if needed

**Medication Not Syncing**:
- Ensure device is online
- Wait a few minutes
- Try editing medication again

**App is Slow**:
- Close other apps
- Clear app cache
- Restart phone

**For More Help**:
- Check Troubleshooting Guide
- Contact support
- Watch tutorial videos
