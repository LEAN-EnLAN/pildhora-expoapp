## States

### Patient States
1. **Supervised (Default)**: Full data sharing with caregivers
2. **Autonomous**: Private mode, no new data shared
3. **Transitioning**: Brief state while mode is being changed

### Caregiver States
1. **Normal View**: Patient in supervised mode, all data visible
2. **Autonomous View**: Patient in autonomous mode, banner shown, no new events
3. **Historical View**: Can always see events from before autonomous mode

## Accessibility

- All switches have proper accessibility labels
- Banner has semantic meaning for screen readers
- Color is not the only indicator (icons + text)
- High contrast between text and background
- Touch targets meet minimum size requirements
