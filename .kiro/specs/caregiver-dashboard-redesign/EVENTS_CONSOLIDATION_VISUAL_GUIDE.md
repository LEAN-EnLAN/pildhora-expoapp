## Performance Features

### Virtualized List
- Only renders visible items
- Smooth 60 FPS scrolling
- Low memory usage

### Real-time Updates
- Firestore onSnapshot listener
- Instant updates when events occur
- No manual refresh needed

### Filter Persistence
- Filters saved to AsyncStorage
- Restored on app restart
- Per-caregiver settings

### Optimized Queries
- Compound Firestore indexes
- Efficient where clauses
- Limited result sets (20 per page)

## User Flows

### View All Events
1. Open Events tab
2. See all medication events in chronological order
3. Pull to refresh for latest data

### Filter by Patient
1. Tap patient filter chip
2. Select patient from modal
3. See only that patient's events
4. Tap "Limpiar" to reset

### Search for Medication
1. Type medication name in search box
2. See real-time filtered results
3. Tap Ã— to clear search

### View Event Details
1. Tap any event card
2. Navigate to event detail screen
3. See full event information
