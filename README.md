# Pildhora

Pildhora es una aplicación móvil diseñada para ayudar a los pacientes a administrar sus medicamentos de manera efectiva. La aplicación se conecta a un dispositivo dispensador de medicamentos inteligente para automatizar el proceso y garantizar que los pacientes tomen sus medicamentos a tiempo.

## Características

- **Gestión de medicamentos:** Agregue, edite y elimine medicamentos de su perfil.
- **Horarios personalizados:** Configure horarios de dosificación personalizados para cada medicamento.
- **Recordatorios inteligentes:** Reciba notificaciones en su teléfono y en el dispositivo dispensador cuando sea el momento de tomar sus medicamentos.
- **Seguimiento del historial:** Mantenga un registro de las dosis tomadas y omitidas.
- **Soporte para cuidadores:** Permita que los cuidadores monitoreen el cumplimiento del tratamiento y reciban alertas.

## Cómo empezar

### Prerrequisitos

- Node.js (versión 18 o superior)
- npm
- Expo CLI

### Instalación

1. Clone el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/pildhora.git
   ```
2. Instale las dependencias:
   ```bash
   npm install
   ```

### Ejecución de la aplicación

1. Inicie el servidor de desarrollo de Expo:
   ```bash
   npm start
   ```
2. Escanee el código QR con la aplicación Expo Go en su dispositivo móvil.
